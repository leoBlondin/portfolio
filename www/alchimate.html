Présentation

Alchimate est une plateforme sur laquelle vous pourrez trouver vos partenaires idéaux pour jouer avec vous ! Ce projet est un exercice du département MMI de Montbéliard.

Ceci est la page promotionnelle pour la plateforme. Vous pouvez y retrouver toutes les informations sur la plateforme, sur ses fonctionnalités ainsi que des informations sur l'équipe de céation.

Ce que j'ai appris

Sur ce projet, j'étais chargé du développement complet de la page promotionnelle. Lors du développement, j'ai par exemple découvert l'utilisation de Display : flex, qui est vite devenu indispensable pour tous mes travaux. J'ai aussi utilisé Javascript et Animate.CSS pour embellir et animer ce joli site ! J'étais également chargé de la création du logo. Ce fut une étape constituée de nombreux essais. Mais nous avons trouvé un terrain d'entente !
